# Paso a paso para levantar el proyecto

1. Clonar el proyecto:

   ```bash
   git clone https://github.com/tonildo123/data-mining.git
   ```

2. Moverse a la carpeta raíz:

   ```bash
   cd data-mining
   ```

3. Instalar dependencias:

   ```bash
   npm install
   ```

   o

   ```bash
   yarn
   ```

4. Ejecutar el proyecto:
   ```bash
   npm start
   ```
